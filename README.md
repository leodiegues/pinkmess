# pinkmess

A completely opinionated PKMS terminal manager for lazy people just like me.


## Installation

```bash
$ pip install git+https://github.com/leodiegues/pinkmess.git
```

## Usage

```bash
$ pinkmess

Usage: pinkmess [OPTIONS] COMMAND [ARGS]...


```
